bool MainWindow::connectToFtp ( const QUrl &  url,
const quint16 &  ftpPort,
const QString &  ftpUser,
const QString  ftpPasswd 
) [private]

Baut Verbindung zum FTP-Server auf und lädt die INI-Datei herunter.

Parameters:
url- Server-URL mit Pfad zur INI-Datei
ftpPort- TCP-Port (default 21)
ftpUser- FTP-Benutzername
ftpPasswd- Passwort des FTP-Benutzers
Returns:
bool - true=Erfolg, false=Fehler

Schließt zunächst bereits geöffnete FTP-verbindung. Erstellt neues QFtp-Objekt, überprüft die angegebene Server-URL, legt eine leere Datei im /tmp-Verzeichnis an, in der die heruntergeladene INI-Datei gespeichert wird.

void MainWindow::loadIniFile ( ) [private]

Lädt die Schlüssel-Wertpaare aus der INI-Datei.
